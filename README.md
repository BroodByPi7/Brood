# Brood

Simple one-page bakery site draft for Brood / BroodByPi7.

## Current direction

- Full front page using `assets/frontpage.png`
- Top-left expanding hover menu
- Scroll sections for Menu, Order, Pickup, Catering, and About
- Daily order and preorder item lists
- No ordering form or calendar yet
